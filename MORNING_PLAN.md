### Block 3 — Traders page: AI-generated leader inference (45 min)

Add a new API endpoint that generates a short NL paragraph about a leader using the data we already have.

**New file: `frontend/app/api/traders/[address]/inference/route.ts`**
```
GET /api/traders/[address]/inference
```
- Reads from DB: `leader_swaps` for this leader (last 100 swaps)
